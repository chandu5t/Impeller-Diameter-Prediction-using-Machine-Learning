# 🏭 About the Project

This project predicts the **optimal pump impeller trim diameter** using
Machine Learning based on industrial pump operating conditions.

It was developed as an **Industry-Sponsored Project** in collaboration
with **Wilo Mather and Platt Pumps Pvt. Ltd.**

The application combines:

-   🤖 Machine Learning
-   ⚡ FastAPI Backend
-   💻 React + Vite Frontend
-   ☁️ Render & Vercel Deployment
-   📊 Interactive Dashboard
-   🔍 Dataset Similarity Search

# ⚙️ Installation

## Clone Repository

``` bash
git clone https://github.com/chandu5t/Impeller-Diameter-Prediction-using-Machine-Learning.git
```

## Backend

``` bash
cd "Wilo 2.0/backend"
python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt
uvicorn main:app --reload
```

## Frontend

``` bash
cd "Wilo 2.0/frontend"
npm install
npm run dev
```
